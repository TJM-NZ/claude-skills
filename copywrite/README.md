# Copywrite

Professional copywriting framework for hero sections, email campaigns, UX microcopy, sales pages, SEO content, and brand tone guides. Generates human-sounding, conversion-focused copy.

## What It Does

Comprehensive copywriting across all marketing touchpoints:

- **Hero Sections** - Headlines, subheadlines, CTAs (benefit/problem/identity-led)
- **Email Campaigns** - Transactional, notification, drip, marketing emails
- **Sales Pages** - Long-form conversion copy, value propositions, feature descriptions
- **UX Microcopy** - Button text, error messages, empty states, tooltips, confirmations
- **SEO Content** - Meta descriptions, page titles, keyword-optimized content
- **Brand Tone** - Voice guides, tone documentation, writing standards

## When to Use

- Writing landing page copy
- Creating email sequences
- Improving conversion rates
- Establishing brand voice
- Optimizing microcopy for UX
- SEO-optimized content creation

## Anti-AI Writing Rules

The copy must read like a human wrote it. Avoid these tells:

**Never use:**
- Emdashes (use commas or periods)
- "unlock", "unleash", "elevate", "empower", "leverage", "streamline"
- "delve", "tapestry", "landscape", "paradigm", "holistic", "synergy"
- "we're thrilled", "we're excited to announce"
- Exclamation marks in subject lines or headlines

**Do use:**
- Contractions ("you'll" not "you will")
- Plain words ("change" not "transform")
- Specific, concrete language
- Active voice, present tense
